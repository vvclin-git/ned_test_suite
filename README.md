# Near Eye Display Test Suite (NETS)

A test suite for near-eye display evaluation.

Main Components:
* Test Pattern Generator
  * Reticle
  * Circle Grid
  * Checkerboard
  * Grille
  * Slanted Edge MTF
* Image Quality Analysis
  * Distortion
  * Grille Contrast
  * Slanted Edge MTF
* Eyebox Calculation using Draper's Method
